Instructions to play the game:


Hit the tile is a very simple iOS game. The user just has to launch the game and click 'Play'. A coloured tile appears of the screen which the user has to remove by tapping on it. If the user taps inside the tile he gets a point and if he taps outside the tile or does not click anywhere then the game is over.
